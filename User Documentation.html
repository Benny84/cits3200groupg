Club Process System User Documentation

What is the Club Process System?

The Club Process System is a way of easily organising and storing information about the running of events combined with a system for sending out email reminders about those events.

Information about the organisation and running of events is stored as “tasks”, which have a title, a body and a status (either inactive, incomplete or complete).

Tasks can then have email reminders set so that the correct person is emailed about the task well in advance of it needing to be completed. Email reminders are composed of a date the email will be sent out on and the recipients email address. Emails are only sent out when tasks are incomplete.

Adding Tasks

To add a completely new task select “+Add Item To Root” from the content pane (located on the far left). Give the task a title and a body of your choosing. These will be included in any emails sent by this task. If you wish to attach an email reminder to the task please check the “Make Task?” box. This will set your tasks status to incomplete by default. Select “Confirm item”.

Tasks can be appended to other tasks by clicking on the task on the content pane and then selecting “+Add Item” in the top right hand corner.

Only incomplete tasks (with the “Make Task?” or "Set As Task?" box checked on task creation) will send out email reminders.

Task Status

If on the creation of a new task you do not select the “Make Task?” box the task will default to being inactive. This means that it will not send out any set email notifications. This status can be changed by using the edit functionality at any time.

If “Make Task?” is checked on creation the task will default to an incomplete task, meaning that it will send email notifications. When the task is completed simply select the check-box next to it to make it a completed task, which will disable email reminders.

Setting Email Reminders

In order to add an email to a Task, we must first add that email to the list of frequently used emails. To do this, select "+ Add/Delete Email" from below the content pane and fill in the add email form.

 

To set an email reminder on your task simply hover over the tasks name and select the letter icon. Fill in the necessary details and click “Set email reminder”. The task will be sent as an email on the morning of the chosen date to the chosen email address.

Removing Tasks and Email Reminders

To remove a task or email reminder hover over it and select the red cross.

Editing Tasks and Email Reminders

To edit a task or email reminder hover over it and select the pencil.

Navigating Around the Hierarchy

Clicking on a task in the content pane will take you to that tasks page and will display all children directly attached to that task. It will not, however, display any tasks attached to those attached tasks. You may click on the names of any attached tasks to navigate to their pages. The content pane will automatically update itself to reflect any changes you make to tasks and emails.
